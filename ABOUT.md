# Athan Alert

**Adhan Alert** is a utility mod for Geometry Dash that helps you keep track of prayer times (Fajr, Dhuhr, Asr, Maghrib, and Isha) while you play. It provides real-time notifications, audio alerts, and a countdown in the pause menu!

## Features

- **AlAdhan API**: Use the AlAdhan API to get the latest prayer times.
- **Real-time Notifications**: Get notified when it's time for prayer
- **Adhan Audio**: Play a customizable Adhan sound when a prayer time is reached. (one is built in!)
- **Offline Support**: Automatically fetches the monthly prayer calendar and saves it, so it works even when you're offline lol
- **Pause Menu**: View the next prayer time directly in the pause menu
- **Today's Schedule**: Access today's full prayer schedule via a button in the main menu
- **Customizable**:
  - Set your **Country** and **City** for accurate times! (GeoIP detection is available too)
  - Choose from **22 Calculation Methods**.
  - Adjust **Adhan Volume** or disable audio entirely.
  - Enable an optional **1-minute reminder** before each prayer.
  - **Auto-pause** levels when it's time to pray.
